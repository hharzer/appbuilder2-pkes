void AppBuilder::refresh ( )

Refreshes the state of AppBuilder, e.g. executing callbacks, connecting to the client.

If AppBuilder isn't connected to a client, but a Serial connection is established, it tries to do the handshake, thus blocking further calculations. Be aware of this fact
